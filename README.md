## About

Babylon.js SPA app with Vite+ stack.

![img](./img/screenshot.png)

## Environment

- Vite+ 0.1.12
- Babylon.js(core) 8.56.0
- Editor: Zed

## Install & Usage

```sh
# install deps
vp install

# run dev server
vp dev

# build
vp build
```
